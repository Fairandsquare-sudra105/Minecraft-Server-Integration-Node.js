## 📋 About Minecraft-Server-Integration-Node.js

This application helps you run a Minecraft server on your Windows PC with no programming skills needed. It works for both Minecraft Java Edition and Minecraft Bedrock Edition. The app offers many options including vanilla support and other server types. It focuses on stability and easy management.

The application uses Node.js and npm modules behind the scenes, but you do not need to know any code to get started. It provides a way to automate your server setup and keep it running smoothly.

## 🖥 System Requirements

- Windows 10 or later (64-bit recommended)
- At least 4 GB of free RAM for Minecraft server
- 5 GB of free storage space for server files and mods
- Stable internet connection for downloads and updates
- Node.js installed (if you want to use advanced options, but this is not needed for normal use)

Most modern Windows PCs meet these specs.

## 🛠 Troubleshooting Tips

- If the server does not start, make sure no other Minecraft server or app is using the same port.
- Confirm your Windows Firewall allows the server app to communicate through the network.
- Restart your computer after installation if you face any errors.
- Make sure you downloaded the correct Windows version of the app.
- For password or connection errors, check that your client Minecraft version matches the server version.

## 📂 Server Folder Structure Explained

When you run the app the first time, it creates a folder with files:

- **server.properties**: Controls the server settings like game mode and player limits.
- **world**: This folder stores the actual Minecraft world data.
- **logs**: Keeps record of server activity and errors.
- **mods** (only if selected): Contains extra features or plugins added to the server.

You can back up or move these folders to keep your world safe.

## 🎯 Supported Minecraft Server Types

- **Vanilla Java Edition**: Official Minecraft server from Mojang.
- **Vanilla Bedrock Edition**: The cross-platform Bedrock engine version.
- **Custom servers**: For mods or special gameplay options.

## 📝 Useful Commands for Server Control

- `stop` – safely shut down the server
- `restart` – reboot the server with minimal downtime
- `status` – check if the server is running and view key info

Use the app interface or Windows terminal to run these commands.
